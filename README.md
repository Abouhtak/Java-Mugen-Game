MUGEN Java Game – Bachelor’s Final Year Project

This project is a 2D fighting game inspired by MUGEN, fully developed in Java as part of my Bachelor’s Final Year Project (Licence PFE).
The project was completed on July 2nd, 2024.

Features

2D character movement (W, A, S, D)

Basic attacks (J)

Enemy AI that tracks and attacks the player

Health bars & win/lose screens

Pause menu & game options

Selectable maps: Konoha, Supmti, Final Valley

Tools & Technologies

Java

Java Swing / AWT (if used)

Custom sprites and assets

Object-oriented game architecture

Purpose

This project was built to demonstrate:

Game loop implementation

Basic collision detection

Animation handling

Simple AI behavior

Menu navigation logic
